# Tykhon Lypnyk — Portfolio

This website showcases graphic design and illustration work by **Tykhon Lypnyk**, prepared as part of a college application. It presents brand identity projects, logo systems, and selected visual work in a focused, editorial layout.

All portfolio images, artwork, and creative direction are **Tykhon's original work**.

**Live site:** [https://lypnyk-tykhon.vercel.app/](https://lypnyk-tykhon.vercel.app/)

## Featured work

- **Duck Club** — Brand identity, logo system, social applications, and process documentation
- **Other works** — Additional graphic design projects, including _Undercover in Miami_

## Tech stack

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Motion](https://motion.dev/) for page transitions and interactions
- [React Router](https://reactrouter.com/)

The site started from a [Figma Make](https://www.figma.com/design/EHBuTiWYZPvIs8FwBv3MdH/Portfolio-Website-for-Designer) export and was refined into a production-ready frontend.

## Development

Frontend implementation, cleanup after Figma-generated code, responsive layout improvements, and deployment were completed with frontend development support.

## Getting started

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal (typically `http://localhost:5173`).

### Build for production

```bash
npm run build
```

Output is written to the `dist/` folder.
